About This Application

This application is an internal tool used exclusively by SET Financial Corporation to:

  • Pull QuickBooks Online financial reports (P&L, Balance Sheet, Cash Flow)
  • Reconcile QBO actuals against Nortridge loan portfolio data
  • Automate month-end journal entry generation
  • Power AI-assisted financial modeling and covenant compliance monitoring

This application is not distributed to or used by third parties.

Disconnect / Revoke Access

To disconnect this application from your QuickBooks Online account, visit QuickBooks Online → Settings → Apps → Connected Apps and revoke access there.
You may also view our disconnect confirmation page.
