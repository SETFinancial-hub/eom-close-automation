# Privacy Policy

**SET Financial Corporation — QuickBooks MCP Integration**

*Last updated: February 19, 2026*

## 1. Introduction

This Privacy Policy describes how the QuickBooks MCP Server application ("Application"), operated by SET Financial Corporation ("Company," "we," "us"), collects, uses, and protects data accessed through the Intuit QuickBooks Online API.

## 2. Data We Access

When authorized, this Application accesses the following data from your QuickBooks Online account:

- Financial reports (Profit & Loss, Balance Sheet, Cash Flow)
- Chart of accounts and general ledger entries
- Invoice and payment records
- Vendor and customer information
- Bank transaction data

## 3. How We Use Your Data

All data accessed through this Application is used exclusively for:

- Internal financial modeling and forecasting
- Portfolio performance analysis
- Cash flow projections and covenant compliance monitoring
- Operational reporting and business intelligence

## 4. Data Storage and Security

- Data is accessed in real-time via the QuickBooks API and is not permanently stored in separate databases unless required for analytical processing.
- All API communications use TLS encryption.
- OAuth 2.0 credentials (client ID, client secret, refresh tokens) are stored as encrypted environment variables, never in source code.
- Access to the Application is restricted to authorized SET Financial Corporation personnel.

## 5. Data Sharing

We do **not** sell, rent, trade, or otherwise share QuickBooks data with any third parties. Data accessed through this Application remains solely within SET Financial Corporation's systems.

## 6. Data Retention

Financial data retrieved from QuickBooks is retained only as long as necessary for the analytical purposes described above. Cached data is refreshed on each API call.

## 7. Your Rights

As the QuickBooks account holder, you may revoke this Application's access at any time through your QuickBooks Online account settings under **Apps** → **Connected Apps**.

## 8. Compliance

This Application complies with:

- Intuit's App Partner Program requirements
- OAuth 2.0 security standards
- Applicable data protection regulations

## 9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date above.
